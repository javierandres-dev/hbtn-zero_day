## Requirements
Install git
```
$ sudo apt-get update
$ sudo apt-get upgrade
$ sudo apt-get install git
```

Basic usage
```
$ git clone <repo>
$ touch test
$ git add test
$ git commit -m "Initial commit"
$ git push origin master
```
